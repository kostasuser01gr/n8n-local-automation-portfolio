# Demo Redaction Checklist

Repository: https://github.com/kostasuser01gr/n8n-local-automation-portfolio

Before recording, verify:

- No `.env` file is visible.
- No credentials are visible.
- No browser session tokens are visible.
- No private local paths are visible.
- No private email address is visible.
- No database password is visible.
- No backup archive path is visible.
- Only public repository pages and sanitized screenshots are shown.
