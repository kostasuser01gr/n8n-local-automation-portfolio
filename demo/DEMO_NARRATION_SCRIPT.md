This is a verified local-first n8n workflow portfolio. It includes three executed workflows with PostgreSQL persistence evidence, screenshots, sanitized workflow exports, and release validation tooling.

The sample data is intentionally small. The counts prove execution and persistence, not scale.

The most important engineering story is the delivery loop: local n8n operation, workflow execution, runtime debugging, PostgreSQL verification, sanitization, and documentation.

The repository is structured so a reviewer can inspect the workflow JSON, schema, sample data, screenshots, and evidence without needing access to the original local environment.
