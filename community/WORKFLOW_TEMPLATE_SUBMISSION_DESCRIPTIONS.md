# Workflow Template Submission Descriptions

Repository: https://github.com/kostasuser01gr/n8n-local-automation-portfolio

## Job Application Intelligence Pipeline

Local-first workflow for importing job opportunities from JSON/CSV, scoring fit, classifying priority, and persisting application records plus import audit metadata to PostgreSQL.

## Fleet Operations Automation System

Deterministic fleet intake workflow that normalizes vehicle records, detects operational priority, upserts current vehicle state, and records processing-run metadata in PostgreSQL.

## Local AI Knowledge Workflow

Local knowledge-processing workflow that cleans text, extracts metadata, classifies documents, applies a review gate, and persists structured knowledge records to PostgreSQL. The verified path does not require paid APIs.
