## Five-Minute Review Path

1. Open the repository README.
2. Review the verified project table.
3. Open the Job Application Intelligence screenshots.
4. Open `docs/EXECUTION_SUMMARY.md`.
5. Open one sanitized workflow JSON.
6. Open `scripts/validate-release.sh` and `scripts/secret-scan.sh`.
7. Review `APPLICATION_READINESS_REPORT.md`.

## Best Evidence Files

- `docs/EXECUTION_SUMMARY.md`
- `case-studies/job-tracker.md`
- `case-studies/fleet-operations.md`
- `case-studies/local-ai-knowledge.md`
- `screenshots/job-tracker/successful-execution-green-nodes.png`
- `REPOSITORY_SECURITY_AUDIT.md`
- `FINAL_RELEASE_REPORT.md`

## What To Look For

- Sanitized workflow exports.
- PostgreSQL schema files.
- Small sample data.
- Authenticated n8n screenshots.
- Validation and secret scan tooling.
- Honest limitation notes.

## What Is Not Included

- Live environment files.
- Database files.
- Backup archives.
- Credential exports.
- Private local runtime material.
