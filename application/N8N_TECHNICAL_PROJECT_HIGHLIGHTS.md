# Technical Project Highlights

Repository: https://github.com/kostasuser01gr/n8n-local-automation-portfolio

## Job Application Intelligence Pipeline

- JSON/CSV intake
- validation and normalization
- fit scoring and priority classification
- PostgreSQL upsert persistence
- import run audit table
- rejection tracking
- local report scripts

## Fleet Operations Automation System

- synthetic fleet intake records
- plate normalization
- duplicate candidate detection
- maintenance/damage/fuel-based priority classification
- PostgreSQL vehicle table
- run summary audit table

## Local AI Knowledge Workflow

- deterministic text cleanup
- metadata extraction
- classification and confidence scoring
- structured JSON generation
- human review gate
- optional disabled Ollama node for local model experimentation

## Infrastructure

- local n8n
- internal PostgreSQL
- external-disk-backed Colima/Docker state
- no named Docker volumes
- no paid API dependency in base workflows
