This portfolio is a local-first automation engineering package built around n8n, PostgreSQL, Docker/Colima, and disciplined workflow documentation.

The strongest project is a Job Application Intelligence Pipeline that ingests local JSON/CSV records, validates them, normalizes fields, scores job fit, classifies priority, writes to PostgreSQL, and maintains import audit/rejection records.

Two additional projects broaden the demonstration:

- Fleet Operations Automation System: realistic rental/fleet intake with duplicate detection, operational priority classification, and audit summaries.
- Local AI Knowledge Workflow: deterministic document processing with optional local Ollama integration for future experimentation.

The emphasis is practical workflow engineering: reproducible local setup, explicit persistence, failure visibility, evidence requirements, and security-aware packaging.
